2. Software Information
To run this project, you will need Python 3.10+. The following packages are required:

pandas: For data manipulation.

wbgapi: To pull live World Bank data.

pyreadstat: To read DHS .dta (Stata) files.

statsmodels: For running the regression analysis.

matplotlib / seaborn: For data visualization.

Installation:
Run the following command in your terminal:


3. Data Acquisition (Manual Steps)
Due to licensing and privacy restrictions, some raw data cannot be hosted directly on GitHub such as the DHS datasets.

A. World Bank Data (Automated)
The script below will automatically download the national-level enrollment and fertility data for Malawi and comparison countries.

```python
Scripts/01_get_wb_data.py
``` 

B. DHS Microdata (Manual Action Required)
Source: The DHS Program.

Action: Register for a research account and request access to the Malawi 2004, 2010, and 2024 Standard DHS datasets.

Download: Obtain the Individual Recode (IR) files in Stata (.dta) format.

Placement: Move the downloaded files into data/raw/.

Note: These files are excluded from GitHub tracking via .gitignore to comply with DHS terms of use.

4. How to Run the Project from Scratch
Follow these steps in order to reproduce the analysis-ready dataset:

Step 1: Run python code/01_get_wb_data.py. This creates data/raw/wb_comparison_raw.csv.

Step 2: Ensure your DHS .dta files are in data/raw/.

Step 3: Run python code/02_clean_dhs.py. This script extracts education levels and birth counts from the survey data.

Step 4: Run python code/03_merge_final.py. This script merges the macro and micro data into the final analysis-ready file located in data/clean/.
